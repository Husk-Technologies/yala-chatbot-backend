# WhatsApp Chatbot Backend API

## Overview

This is the backend API for the **WhatsApp Chatbot** that integrates with **WhatsApp Cloud API**. The API handles event information, guest interactions, condolences, donations, and brochure management.

## Prerequisites

Before setting up the project, ensure you have the following installed:

* Node.js (v14 or higher)
* A database (MongoDB or PostgreSQL)
* WhatsApp Cloud API access

## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/Husk-Technologies/yala-chatbot-backend.git
   cd whatsapp-chatbot-backend
   ```
